# 最长公共子序列

  • str1 的长度为 m，str2 的长度为 2
  • dp[i][j] 表示长度为 i 的 str1 子串和长度为 j 的 str2 子串的最长公共子序列的长度。子串取 [0:i) 或 [0:j) 范围的字符串。
  • i 和 j 均从 0 开始增长。
if (str1[i]==str[j])
	dp[i][j] = dp[i-1][j-1] + 1;
else
	dp[i][j] = max(dp[i][j-1], dp[i-1][j]);
