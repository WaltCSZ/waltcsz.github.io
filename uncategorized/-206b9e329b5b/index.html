# 栈迁移

栈迁移大致在以下情况使用：

  • 栈空间太小不足以用来攻击
  • 为了利用缓冲区溢出攻击某些地址（待验证）
  • 迁移至.bss 段上确保权限足够

函数返回时将栈中旧 ebp 弹入 ebp 寄存器，而对旧 ebp 进行伪造可以将栈底迁移至构造的 fake_stage，再次利用 ROP 执行 leave, ret 可以将 ebp 弹入 esp，实现完整的栈迁移。
